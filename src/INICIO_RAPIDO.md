# ⚡ Inicio Rápido - 3 Minutos

## 🎯 Solo 4 Comandos

### **1. Abrir Terminal en VS Code**
```
Presiona: Ctrl + `
```

### **2. Instalar Dependencias** (solo primera vez)
```bash
npm install
```

### **3. Ejecutar Proyecto**
```bash
npm run dev
```

### **4. Abrir Navegador**
```
http://localhost:5173/
```

**¡LISTO!** 🎉

---

## 🔄 Uso Diario

Cada vez que trabajes en el proyecto:

```bash
# 1. Abrir terminal (Ctrl + `)
# 2. Ejecutar

# 3. Abrir navegador
# http://localhost:5173/
```

## 🛑 Detener Servidor

```
Ctrl + C  (en la terminal)
```

---

## 📱 Probar Responsive

1. **F12** (abrir DevTools)
2. **Ctrl + Shift + M** (modo responsive)
3. **Seleccionar dispositivo** (iPhone, iPad, etc.)

---

## ❌ Error Común

**"command not found: npm"**

→ Instala Node.js desde: https://nodejs.org/

---

## 💡 Tips

### Múltiples Terminales
```
Ctrl + Shift + `  (nueva terminal)
```

### Limpiar Terminal
```bash
clear      # Mac/Linux
cls        # Windows
```

### Ver Cambios en Vivo
Los cambios se reflejan automáticamente al guardar (Ctrl + S)
